```scss /// Примесь, которая выводит вендорные префиксы /// @access public /// @author KittyGiraudel /// @param {String} $property - свойство CSS без префикса /// @param {*} $value - Сырое значение свойства CSS /// @param {List} $prefixes - Список префиксов для вывода @mixin prefix($property, $value, $prefixes: ()) { @each $prefix in $prefixes { -#{$prefix}-#{$property}: $value; } #{$property}: $value; } ```
