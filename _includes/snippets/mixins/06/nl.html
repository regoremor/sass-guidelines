```scss /// Mixin-helper om voorvoegsels van leveranciers uit te voeren /// @access public /// @author KittyGiraudel /// @param {String} $property - CSS-eigenschap zonder prefix /// @param {*} $value - Ruwe CSS-waarde /// @param {List} $prefixes - Lijst met voorvoegsels die moeten worden uitgevoerd @mixin prefix($property, $value, $prefixes: ()) { @each $prefix in $prefixes { -#{$prefix}-#{$property}: $value; } #{$property}: $value; } ```
